BalcanCraft Staff API

Todo lo que necesitas saber para poder ayudar a los jugadores de BalcanCraft




Lobby || Survival || Otros


Lobby

  • /skin <nombre_jugador_premium>
    • Sirve para cambiarse de skin, se tiene que volver a entrar al servidor - SkinRestorer

  • /spawn
    • Permite a la gente volver al spawn del servidor - EssentialsX

  • /fly
    • Permite sólo a la gente VIP volar por el lobby - EssentialsX

Survival

  • /skin <nombre_jugador_premium>
    • Sirve para cambiarse de skin, se tiene que volver a entrar al servidor - SkinRestorer

  • /spawn
    • Permite a la gente volver al spawn del servidor - EssentialsX

  • /fly
    • Permite sólo a la gente VIP volar por el servidor - EssentialsX

  • /warps
    • Permite abrir el menú de warps para viajar más fácilmente por el servidor - EssentialsX y ChestCommands

  • /sethome
    • Permite tener un "hogar" establecido - EssentialsX

  • /home
    • Permite teletransportarse al "hogar establecido" - EssentialsX
